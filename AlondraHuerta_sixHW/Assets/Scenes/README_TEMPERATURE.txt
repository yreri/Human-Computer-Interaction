This code to be sucessfully run, needs the user to make sure of this specific requirements:

Inside the menu of Project Settings -> Player Settings the options
	-Player -> Configuration -> Api Compability Level* .NET 4.x
	-Player -> Resolution and presentation -> Resolution -> Fullscreen Mode -> Windowed
	-Player -> Resolution and presentation -> Resolution -> Default Screen Width -> 540
	-Player -> Resolution and presentation -> Resolution -> Default Screen Height -> 640
	-The Display has to be 540x640
