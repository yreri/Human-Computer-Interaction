This code to be sucessfully run, needs the user to make sure of this specific requirements:

Inside the menu of Project Settings -> Player Settings the options
	-Fullscreen Mode -> Fullscreen Window
	-Run In Background(enable)
	-Visible in Background(enable)
	-Allow Fullscreen(enable)
	-Use DXGI Flip Model Swapchain for D3D11(disable)


Inside the menu of Project Settings -> Graphics the options
	-Scriptable Render Pipeline -> None

In addition it needs to be build and not only play
